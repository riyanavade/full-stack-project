# Capstone Project

This repository contains a full-stack capstone project (backend + frontend).

Quick start:

- Backend: install dependencies and run server
  - `cd backend`
  - `npm install`
  - `node server.js`

- Frontend: install dependencies and run dev server
  - `cd frontend`
  - `npm install`
  - `npm run dev`

Notes:
- `.env` files are ignored and should not be committed.
